# Mini GoResto

## Description

An Application to provide a menu consisting of name, description, and price.

## Environment Setup

Dependancies:

* Ruby
* Rails
* Bundle

To setup environment, run:

```sh
bundle install
```

## Build Instruction

To prepare database, run migration by:

```sh
rails db:migrate
```

## Run Instruction

To run tests, run:

```sh
bundle exec rspec ./spec/*
```

To start server, run:

```sh
rails s
```
